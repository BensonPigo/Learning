

### 1\. 高層次概念圖 (洋蔥圖)
這張圖展示了架構的核心規則：所有依賴關係都指向Core。

```plaintext
+--------------------------------------------------------------------------+
|   UI Layer / API Layer (例如：WinForms 的 OrderForm.cs)                  |
|   +---------------------------------+                                    |
|   |         OrderForm.cs            |                                    |
|   |---------------------------------|    (依賴/使用)                     |
|   | 使用 IOrderService (菜單)       | ----------------------------------> |
|   | 建立 PlaceOrderCommand          |                                    |
|   +---------------------------------+                                    |
+--------------------------------------------------------------------------+
        |
        |  <<傳遞 PlaceOrderCommand>>
        v
+--------------------------------------------------------------------------+  <-- 依賴方向
|   Application Layer (大腦)                                               |
|   +-----------------------------+     +-------------------------------+ |
|   |     IOrderService.cs        |     |         OrderService.cs       | |
|   |-----------------------------|     |-------------------------------| |
|   | 【對外的介面 - 菜單】       |     | 實作 IOrderService            | |
|   | 定義應用程式能做什麼        |     |                               | |
|   +-----------------------------+     | (依賴/使用)                   | |
|                                       |-------------------------------> |
|                                       | IOrderRepository (插座)       | |
|                                       +-------------------------------+ |
+--------------------------------------------------------------------------+
        |
        |  <<呼叫 Save() 方法>>
        v
+--------------------------------------------------------------------------+  <-- 依賴方向
|   Core Layer (心臟)                                                      |
|   +-----------------------------+     +-------------------------------+ |
|   |         Order.cs            |     |     IOrderRepository.cs       | |
|   |-----------------------------|     |-------------------------------| |
|   | 包含核心業務規則 (Entities) |     | 【對內的介面 - 插座】         | |
|   |                             |     | 定義 Core 需要什麼持久化服務  | |
|   +-----------------------------+     +-------------------------------+ |
|                                                                          |
|               (最核心，不依賴任何東西)                                   |
+--------------------------------------------------------------------------+
        ^
        |  <<實作 (Implements)>>
        |
+--------------------------------------------------------------------------+
|   Infrastructure Layer (手腳)                                            |
|   +-----------------------------+                                        |
|   |    SqlOrderRepository.cs    |                                        |
|   |-----------------------------|                                        |
|   | 實作 IOrderRepository       |                                        |
|   | 包含具體的 SQL 程式碼       |                                        |
|   +-----------------------------+                                        |
+--------------------------------------------------------------------------+
```
